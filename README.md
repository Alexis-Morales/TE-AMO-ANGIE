# TE-AMO-ANGIE
pagina para angie bb

Instrucciones rápidas

- Coloca una fuente pixelada (opcional) llamada `minecraftia.ttf` en `assets/fonts/`.
	- Si no tienes la fuente, la página usará `monospace` como alternativa.
- Abrir localmente:

```bash
cd /workspaces/TE-AMO-ANGIE
python3 -m http.server 8000
# Abrir http://localhost:8000 en tu navegador
```

Editar contenido

- `index.html`: contenido de la página y galería.
- `css/styles.css`: estilos y la referencia a la fuente.

Si quieres que yo añada más imágenes a la galería o incluya la fuente directamente (si me das el archivo), dímelo y la subo por ti.
